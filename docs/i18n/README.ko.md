# Simplicio Canvas

[English README](../../README.md) · 한국어

Three.js 기반의 읽기 우선 소프트웨어 아키텍처 캔버스입니다. 프로젝트 파일을 퍼즐 조각과 연결선으로 보여 주며, 확대 수준에 따라 프로젝트·흐름·파일과 관계를 탐색할 수 있습니다.

기본 예제는 [`wesleysimplicio/simplicio-loop`](https://github.com/wesleysimplicio/simplicio-loop)의 안전한 스냅샷입니다. 예정된 bootstrap은 `simplicio-mapper`와 로컬 `simplicio-loop` 스킬도 준비합니다.

## 실행

```bash
npm install
npm run dev
```

MVP: 로컬 폴더, 언어와 import, 이동 가능한 조각, 3D 탐색, Explorer, 탭, 터미널/편집기, Inspector 및 반응형 화면을 제공합니다. 코드는 로컬에 남습니다.
