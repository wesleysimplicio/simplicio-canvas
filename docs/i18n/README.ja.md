# Simplicio Canvas

[← English](../../README.md) · **日本語**

ローカルファーストのビジュアルプログラミング。フォルダーを開くと、プロジェクト、レイヤー、フロー、ファイル、import を VS Code 風の Three.js Canvas 上の接続された部品として確認できます。

デフォルト例は [`wesleysimplicio/simplicio-loop`](https://github.com/wesleysimplicio/simplicio-loop) の安全なスナップショットです。予定されているブートストラップでは `simplicio-mapper` とローカルの `simplicio-loop` スキルも準備します。

```bash
git clone https://github.com/wesleysimplicio/simplicio-canvas.git
cd simplicio-canvas && npm install && npm run dev
```

MVP：ローカルフォルダー、言語と import、移動可能な部品、3D ナビゲーション、Explorer、タブ、terminal/editor、Inspector、レスポンシブ表示。コードはローカルに留まります。
